MSR Challenge Submission Format

Overview

Your submission must be a .zip file containing exactly eight stem folders, each corresponding to one instrument/vocal type. Each folder must contain exactly 125 WAV files (one per clip), named according to the provided file list (e.g., clip_0000.wav, clip_0001.wav, ...).

Important: The folder names, file counts, and filenames must exactly match the reference data. Missing or extra files will cause the scoring to fail.

Expected Folder Structure

submission.zip
├── bass
│   ├── clip_0000.wav
│   ├── clip_0001.wav
│   ├── ...
│   └── clip_0124.wav
├── drums
│   ├── clip_0000.wav
│   ├── ...
│   └── clip_0124.wav
├── guitars
│   ├── clip_0000.wav
│   └── ...
├── keyboards
│   ├── clip_0000.wav
│   └── ...
├── orchestral
│   ├── clip_0000.wav
│   └── ...
├── percussion
│   ├── clip_0000.wav
│   └── ...
├── synthesizers
│   ├── clip_0000.wav
│   └── ...
└── vocals
    ├── clip_0000.wav
    └── ...

Stem Folders (Must Match Exactly)

  • bass
  • drums
  • guitars
  • keyboards
  • orchestral
  • percussion
  • synthesizers
  • vocals

File Format Requirements

  • Format: 16-bit PCM WAV
  • Sample rate: 44.1 kHz or higher (scoring will resample as needed)
  • Mono or stereo — match original stem characteristics
  • File names must exactly match the reference names (case-sensitive)

Packaging Your Submission

Do NOT zip the folder containing your stems. Instead, zip the stems themselves from the submission root directory. This ensures that the root of the zip contains the 8 stem folders directly, not an extra folder level.

✅ Correct way:

cd /path/to/submission_root
zip -r submission.zip bass drums guitars keyboards orchestral percussion synthesizers vocals

❌ Incorrect way (will create an extra folder level inside the zip):

cd /path/to
zip -r submission.zip submission_root

Submission Checklist

  • ✅ The .zip contains exactly 8 folders (one for each stem)
  • ✅ Each folder contains exactly 125 files
  • ✅ All filenames match the provided reference list (clip_0000.wav → clip_0124.wav)
  • ✅ No missing or extra files
  • ✅ No nested subfolders inside stem folders
  • ✅ All files are valid WAV format and loadable
